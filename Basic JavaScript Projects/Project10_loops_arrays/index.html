While Loops

A while loop is basically a repeating “if statement.” Meaning, you are telling the computer

to execute certain code repeatedly while a particular condition is present.

For Loops

A “For Loop” is used to repeat a section of code a number of times. For Loops are used

when the number of iterations are known.

Arrays and Objects

An array is a collection of data, arranged in rows and columns.

Arrays are objects and so are included in the object data type. Objects can have properties

(characteristics) and methods (actions). Arrays are a special type of object.

Const Keyword

The const keyword creates a constant whose scope can be either global or local to the

block in which it is declared.

Let Keyword

The let keyword declares variables that can have block scope. This refers to blocks of code contained

within curly brackets — commonly loops.

The return Statement

The return statement stops the execution of a function and returns a value.

E.g.: Call a function that returns the value of PI
